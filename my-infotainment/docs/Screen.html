Screen(props, screenBrightness, showMessage, showBrightnessSlider, showSysMenu, network, trainData, loading, lastDigitOfServer, response, responseColor, lazy)

new Screen(props, screenBrightness, showMessage, showBrightnessSlider, showSysMenu, network, trainData, loading, lastDigitOfServer, response, responseColor, lazy)

A functional component which is responsible for displaying our screen user interface with its sub components childrens such as Message, trainDataForm and softkey

Name Type Description

showMessage boolean A state variable responsible for conditional rendering of the message component
showBrightnessSlider boolean A state variable responsible for conditional rendering of Brighness slider
showSysMenu boolean A state variable responsible for conditional renering the system menu
network string A state variable storage the current network mode
trainData object A state variable storage the train data num,Eva, typ
loading boolean A state variable responisble for conditional rendering of loading component the animation
lastDigitOfServer int A state variable storage the last digit of the flask web servers it could be 0,1,2,3
response string A state variable storage the response status text of the flask webserver
responseColor string A state variable color the response of the flask web server green if server recieved message successfully or red if fails
lazy function A built-in function responsible for lazy importing of some of react components. good for performance optimizing
